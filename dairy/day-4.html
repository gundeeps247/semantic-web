Day 4

There are standards used to represent relationships with data and the web.

Example - Swiggy:
- One person orders food.
- One person makes the food (unknown to the one ordering).
- A delivery person is unfamiliar with both.

All three are in relationships.

RDF - Triple (Graph):
A general framework for representing interconnected data on the web.

RDF statements describe metadata, allowing standardized data exchange based on relationships. RDF integrates data from multiple sources.

Why Study RDF?
It's fast and efficient.

RDF Full Form:
Resource Description Framework

Types of Data Representations:
- Document relationship
- People
- Concept
- Data object

Triple Components:
- Subject
- Predicate
- Object

Metadata Exchange:
Exchange of data between users and the internet.

User Behavior Analysis:
Metadata helps identify user needs based on their interactions (e.g., personalized web options based on search history).

Semantic Metadata:
Describes the meanings of values for human or programmatic use.

Triple Breakdown:
- Subject: A resource or concept.
- Object: Value of the predicate.
- Predicate: A property or attribute linking subject and object.
